Sur la classification en blocs

Les blocs ne sont pas des groupes parlementaires officiels. Ce sont des regroupements politiques de fait, basés sur les alliances électorales 2024 (NFP), les soutiens parlementaires publics du gouvernement, et la position institutionnelle (charnières). Le mapping est documenté dans la page À propos.

Diagramme ternaire des groupes politiques

Chaque groupe est positionné dans un triangle dont les 3 sommets représentent 3 ancres (par défaut les 3 blocs principaux). La position d'un groupe reflète ses cohésions de vote relatives avec ces 3 ancres : plus il est proche d'un sommet, plus il vote comme l'ancre de ce sommet.

Défaut : {{ tri.anchor_top_label }} (haut), {{ tri.anchor_bl_label }} (bas-gauche), {{ tri.anchor_br_label }} (bas-droite)

Comment lire un diagramme ternaire

  • Les 3 sommets sont les 3 ancres. Un point placé exactement sur un sommet = 100 % de cohésion avec cette ancre. Plus on s'éloigne du sommet, moins on en partage les votes.
  • Le centre du triangle = équilibre parfait entre les 3 ancres (33 % chacune). C'est typiquement la zone des groupes pivots.
  • Les côtés représentent les axes opposés à chaque sommet. Plus un point se rapproche d'un côté, plus il s'éloigne du sommet opposé.
  • Survole un cercle pour voir ses 3 cohésions exactes (en %).
  • Les positions sont calculées en coordonnées barycentriques à partir de la matrice de cohésion : pour chaque groupe, ses 3 cohésions avec les 3 ancres sont normalisées pour sommer à 100 %.
