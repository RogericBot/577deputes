Méthodologie & RGPD

Détection bots : un visiteur est classé « bot » si son User-Agent matche un pattern connu (Googlebot, Bingbot, Claude, GPT, Wget, Curl, etc.) OU s'il ne contient pas la chaîne Mozilla (heuristique : tous les navigateurs réels la mettent ; son absence indique un script ou outil). UA vide = bot suspect.

Top User-Agents : seules des catégories agrégées sont stockées (ex : « Chrome desktop », « Bot — Googlebot »). Aucun User-Agent brut, aucune IP, aucun fingerprint n'est conservé.

Top pages : les UID sont remplacés par [uid] pour agréger (ex : /deputes/[uid] au lieu de /deputes/PA12345). Pas de query string conservée.

Données stockées : data/analytics.db, jamais transmis à un tiers, jamais exporté, accessible uniquement depuis cette page (HTTP Basic Auth).
